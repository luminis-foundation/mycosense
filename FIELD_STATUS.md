# MycoSense Field Status

MycoSense is an active Luminis Foundation research software and hardware project for observing ecological and mycelial signal patterns in high-desert root-zone environments.

## Current Status

The public dashboard currently runs in simulated or mock data mode.

The software stack includes:

- React and Vite dashboard
- mock sensor stream
- DSP signal-processing utilities
- zone health scoring
- microclimate and weather display layer
- Raspberry Pi gateway scaffold
- ESP32-C6 and ESP32-S3 firmware scaffold
- data export and provenance hashing utilities

## Live Hardware Status

Live hardware integration is designed but not yet field-verified.

The intended live stack includes:

- ESP32-C6 or ESP32-S3 sensor nodes
- electrode-based mycelial and root-zone signal sensing
- environmental sensors for temperature, humidity, pressure, light, wind, and rain
- Raspberry Pi coordinator or gateway
- local data logging and synchronization
- future public dataset publication after validation

## Field Deployment Status

First field deployment is pending.

No public claim should describe MycoSense as field-deployed until physical hardware has been purchased, flashed, tested, installed, and validated in the Pecos watershed or Northern New Mexico field environment.

## Funding Dependency

Hardware verification and field deployment depend on available Luminis Foundation budget for components, sensor boards, microcontrollers, power systems, enclosures, and field-testing materials.

## Public Communication Rule

Use these terms precisely:

- "Simulated dashboard" for current public demo mode.
- "Hardware integration path" for firmware and Pi gateway work.
- "Pending field deployment" for live environmental deployment.
- "Field-verified" only after real hardware has collected validated live data.
